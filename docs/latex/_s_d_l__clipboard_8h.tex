Get UTF-\/8 text from the clipboard, which must be freed with \mbox{\hyperlink{_s_d_l__stdinc_8h_a0e4ade894d550ada1fa19dc2d46e88b8}{SDL\+\_\+free()}}.

This functions returns empty string if there was not enough memory left for a copy of the clipboard\textquotesingle{}s content.

\begin{DoxyReturn}{Returns}
the clipboard text on success or an empty string on failure; call \mbox{\hyperlink{_s_d_l__error_8h_adaef317de186f072f8328798de29d51a}{SDL\+\_\+\+Get\+Error()}} for more information. Caller must call \mbox{\hyperlink{_s_d_l__stdinc_8h_a0e4ade894d550ada1fa19dc2d46e88b8}{SDL\+\_\+free()}} on the returned pointer when done with it (even if there was an error).
\end{DoxyReturn}
