\doxysubsection{Detailed Description}
The bits of this structure can be directly reinterpreted as an integer-\/packed color which uses the SDL\+\_\+\+PIXELFORMAT\+\_\+\+RGBA32 format (SDL\+\_\+\+PIXELFORMAT\+\_\+\+ABGR8888 on little-\/endian systems and SDL\+\_\+\+PIXELFORMAT\+\_\+\+RGBA8888 on big-\/endian systems).
