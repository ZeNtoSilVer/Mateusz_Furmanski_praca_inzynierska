Accelerometer sensor

The accelerometer returns the current acceleration in SI meters per second squared. This measurement includes the force of gravity, so a device at rest will have an value of SDL\+\_\+\+STANDARD\+\_\+\+GRAVITY away from the center of the earth.

values\mbox{[}0\mbox{]}\+: Acceleration on the x axis values\mbox{[}1\mbox{]}\+: Acceleration on the y axis values\mbox{[}2\mbox{]}\+: Acceleration on the z axis

For phones held in portrait mode and game controllers held in front of you, the axes are defined as follows\+: -\/X ... +X \+: left ... right -\/Y ... +Y \+: bottom ... top -\/Z ... +Z \+: farther ... closer

The axis data is not changed when the phone is rotated.

Gyroscope sensor

The gyroscope returns the current rate of rotation in radians per second. The rotation is positive in the counter-\/clockwise direction. That is, an observer looking from a positive location on one of the axes would see positive rotation on that axis when it appeared to be rotating counter-\/clockwise.

values\mbox{[}0\mbox{]}\+: Angular speed around the x axis (pitch) values\mbox{[}1\mbox{]}\+: Angular speed around the y axis (yaw) values\mbox{[}2\mbox{]}\+: Angular speed around the z axis (roll)

For phones held in portrait mode and game controllers held in front of you, the axes are defined as follows\+: -\/X ... +X \+: left ... right -\/Y ... +Y \+: bottom ... top -\/Z ... +Z \+: farther ... closer

The axis data is not changed when the phone or controller is rotated.
