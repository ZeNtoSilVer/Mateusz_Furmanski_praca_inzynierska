The SDL virtual key representation. 

Values of this type are used to represent keyboard keys using the current layout of the keyboard. These values include Unicode values representing the unmodified character that would be generated by pressing the key, or an SDLK\+\_\+$\ast$ constant for those keys that do not generate characters.

A special exception is the number keys at the top of the keyboard which always map to SDLK\+\_\+0...SDLK\+\_\+9, regardless of layout.
