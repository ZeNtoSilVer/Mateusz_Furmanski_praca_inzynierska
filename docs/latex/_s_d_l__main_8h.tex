Circumvent failure of \mbox{\hyperlink{_s_d_l_8h_a8fc8d35348d7c74bad8392d776c937b8}{SDL\+\_\+\+Init()}} when not using \mbox{\hyperlink{_s_d_l__main_8h_a35282508ae396e1fd5711ff3d85e08a7}{SDL\+\_\+main()}} as an entry point.

This function is defined in \mbox{\hyperlink{_s_d_l__main_8h}{SDL\+\_\+main.\+h}}, along with the preprocessor rule to redefine \mbox{\hyperlink{main_8cpp_a700a0caa5b70a06d1064e576f9f3cf65}{main()}} as \mbox{\hyperlink{_s_d_l__main_8h_a35282508ae396e1fd5711ff3d85e08a7}{SDL\+\_\+main()}}. Thus to ensure that your \mbox{\hyperlink{main_8cpp_a700a0caa5b70a06d1064e576f9f3cf65}{main()}} function will not be changed it is necessary to define SDL\+\_\+\+MAIN\+\_\+\+HANDLED before including \mbox{\hyperlink{_s_d_l_8h}{SDL.\+h}}.
