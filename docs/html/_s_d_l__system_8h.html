DECLSPEC SDL_bool SDLCALL SDL_IsTablet ( void  )

Query if the current device is a tablet.

If SDL can't determine this, it will return SDL_FALSE.

Returns
SDL_TRUE if the device is a tablet, SDL_FALSE otherwise.
Since
